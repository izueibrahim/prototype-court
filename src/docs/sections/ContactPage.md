# ContactPage Documentation

The `ContactPage.tsx` section facilitates communication between the public and different court departments.

## Features
- **Department Directory**: Contact details for various units (E-filing, C.A. Unit, Registry).
- **Inquiry Form**: Standardized form for public queries.
- **Map & Location**: Visual representation of the court's physical address.

## UI Characteristics
- **Style**: Grid-based layout for contact cards.
- **Accessibility**: Clear labels and high-contrast support.

## Usage
```tsx
import { ContactPage } from '@/sections/ContactPage';

// Rendered when 'Contact' is selected in the portal
<ContactPage />
```
