# HeroSection Section

The primary entry point of the portal featuring the main brand message and global search.

## Features
- **Visual Impact**: Uses bold typography and animated "Version 2.0" indicators.
- **Global Search**: A prominent search input that redirects the user to the dedicated Search Results page.
- **Glassmorphism**: Styled with modern backdrop blurs and subtle gradients.

## Technical Details
- **File**: `src/sections/HeroSection.tsx`
- **Navigation**: Submission of the search form sets `currentView` to 'search' and updates `searchQuery` in the store.
