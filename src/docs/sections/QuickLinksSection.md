# QuickLinksSection Section

A horizontal scrollable strip of common utility actions.

## Features
- **Action Cards**: Quick access to "File a Document", "Join Virtual Court", etc.
- **Centered Layout**: Items are centered on the screen for better balance when fewer items are present.
- **Carousel Navigation**: Supports horizontal swiping/scrolling on all devices.

## Technical Details
- **File**: `src/sections/QuickLinksSection.tsx`
- **Alignment**: Uses `justify-center` for layout centering.
