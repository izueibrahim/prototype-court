# EServicesDirectory Section

Explores the comprehensive suite of online judicial services.

## Features
- **Tabbed Browsing**: Categorizes services into groups like "Public Access" and others.
- **Carousel UI**: Services are displayed in a horizontal carousel for easy scanning.
- **Data-Driven**: Content is populated from `src/lib/data.ts`.

## Technical Details
- **File**: `src/sections/EServicesDirectory.tsx`
- **State**: Uses local state to track the active category tab.
