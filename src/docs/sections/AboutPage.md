# AboutPage Documentation

The `AboutPage.tsx` section provides public information about the Industrial Court of Malaysia, its mission, and its organizational structure.

## Features
- **Mission & Vision**: Clear presentation of the court's objectives.
- **Structure**: Overview of the judicial hierarchy and administration.
- **Rich Media**: Utilizes generated imagery to represent the court's professionalism.

## UI Characteristics
- **Style**: Informative layout with readable typography.
- **Consistency**: Follows the global portal design system.

## Usage
```tsx
import { AboutPage } from '@/sections/AboutPage';

// Rendered when 'About Us' is selected in the portal
<AboutPage />
```
