# Root Layout

The foundational layout wrapper for the Next.js eMP application.

## Responsibility
- Defines the `<html>` and `<body>` structure.
- Configures global fonts and SEO metadata.
- Initializes Tailwind CSS via global imports.

## Technical Details
- **File**: `src/app/layout.tsx`
- **Assets**: Imports `@/app/globals.css`.
