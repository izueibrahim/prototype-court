# e-Mahkamah Perusahaan (eMP) v2.0 Prototype

A modern, responsive web application prototype for the Industrial Court of Malaysia, focusing on digitizing case management, e-filing, and virtual hearings (e-Sebutan).

## Core Modules

- **Internal Dashboard**: Comprehensive workspace for Court Officers, Chairmen, and the YDP. Features advanced analytics, case allocation, and automated scheduling.
- **e-Filing Portal**: Secure document submission and case tracking for external parties.
- **Guest Dashboard (e-Sebutan)**: Simplified access to virtual hearings with integrated video conferencing and automated admission logic.
- **MyDigital ID Integration**: Secure SSO authentication using the national digital identity platform.

## Key UI/UX Features

- **Standardized Component Library**: Unified design system across all modules.
- **Dynamic Case Steppers**: Real-time visual tracking of case progress from registration to award.
- **Advanced Reporting Filters**: Intuitive data exploration with smart defaults (Year 2026) and 4-digit formatting.
- **Mobile-Responsive Virtual Rooms**: Optimized participant grids for seamless virtual hearing attendance on any device.

## Technology Stack

- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Icons**: Lucide React
- **Branding**: Malaysian Industrial Court Standard

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
